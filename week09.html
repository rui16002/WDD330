The Screen object has more uses for mobile devices. It also allows you to do things like turn off the device’s screen, detect a change in its orientation or lock it in a specific orientation.

Cookies

Cookies can be used for personalizing a user’s browsing experience, storing user preferences, keeping track of user choices (such as a shopping cart), authentication and tracking users. The use of cookies for tracking purposes has been much maligned in recent years. Their use for data storage is starting to be replaced in many cases by the new HTML5 localStorage API as it allows more data to be stored.

we should not rely on the cookie expires value

timing functions

setTimeout will call a function after certaint time

setInterval is similar but it will keep calling the callback function.

These two methods could be used to perform animations although the requestAnimationFrame should be used instead for better performance. CancelAnimation frame is used to stop the animation. CSS should be used intead of these functions whenever posible.

HTML5

The data- atribute could be used to set attributes to elements. values could be access like: element.dataset.attribute

The HTML5 specification contains a number of APIs that help to gain access to hardware, such as cameras, batteries, geolocation, and the graphics card

localStorage works in a similar way as cookies, but there is virtualy no storage limit. sessionStorage works in a similar fashion but data expires with the session.

Geolocation

We can get the device's location like: navigator.geolocation.getCurrentPosition(youAreHere);. youAreHere is a callback aparameter is passed to this callback this parameter has a coords property with a latitude and longitud. there are several other properties like, position.altitude, speed, heading, timestamp
